    [List the types of data collected through cookies, analytics, or other tracking technologies]
    We collect the following information automatically:
    - Your IP address (This is not linked to any account, and is only used for API rate-limiting/spam protection purposes)
    - Your detected operating system (This is not linked to any IP address or account, and is only used for anonymous web analytics)

    - Your detected device brand (E.g: iOS, Android, Windows, MacOS, Linux) (This is not linked to any IP address or account, and is only used for anonymous web analytics)
