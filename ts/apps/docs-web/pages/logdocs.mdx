# NootBox Logging Documentation

 - `WIFI`: Anything to do with a wireless network connection
 - `FLSH`: Anything to do with the flash memory
 - `HTTP`: Anything to do with sending or receiving HTTP requests
 - `CHSE`: Anything to do with the `checkErrorStorage` function
 - `INFO`: Anything to do with printing informational messages
 - `ERRO`: Anything to do with printing error messages
 - `WARN`: Anything to do with printing warning messages
 - `OTHE`: Anything OTHER THAN the above
